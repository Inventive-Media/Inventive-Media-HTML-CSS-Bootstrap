Day 1 - Block Elements

Block-level elements are elements that take up the full width of their parent container.

1. Headings

Headings are defined with the <h1> to <h6> tags.

Code:

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>

Output:

This is heading 1

This is heading 2

This is heading 3

This is heading 4

This is heading 5
This is heading 6

2. Paragraphs

Paragraphs are defined with the <p> tag.

Code:

<p> Lorem ipsum dolor sit amet consectetur adipisicing elit. Distinctio, ratione. </p>

Output:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Distinctio, ratione.

3. Preformatted Text

The <pre> tag preserves both spaces and line breaks.

Code:

<pre> Lorem ipsum dolor sit amet consectetur, adipisicing elit. Dolore, itaque! </pre>

Output:

Lorem            ipsum dolor sit amet consectetur, 
adipisicing elit. Dolore, itaque!

4. Ordered Lists

Ordered lists usage with different type attributes.

Code (Default - Numbers):

<h3>Milktea Menu</h3> <ol type="1"> <li>Macha</li> <li>Taro</li> <li>Chocolate</li> </ol>

Output:

Milktea Menu

  1. Macha
  2. Taro
  3. Chocolate

Code (Uppercase Letters):

<ol type="A"> <li>Macha</li> <li>Taro</li> <li>Chocolate</li> </ol>

Output:

  1. Macha
  2. Taro
  3. Chocolate

Code (Lowercase Roman Numerals):

<ol type="i"> <li>Macha</li> <li>Taro</li> <li>Chocolate</li> </ol>

Output:

  1. Macha
  2. Taro
  3. Chocolate

Code (Uppercase Roman Numerals):

<ol type="I"> <li>Macha</li> <li>Taro</li> <li>Chocolate</li> </ol>

Output:

  1. Macha
  2. Taro
  3. Chocolate

5. Unordered Lists

Unordered lists usage with different type attributes.

Code (Circle):

<h3>Soda's</h3> <ul type="circle"> <li>Coke</li> <li>Pepsi</li> <li>Sarsi</li> </ul>

Output:

Soda's

  • Coke
  • Pepsi
  • Sarsi

Code (Square):

<ul type="square"> <li>Coke</li> <li>Pepsi</li> <li>Sarsi</li> </ul>

Output:

  • Coke
  • Pepsi
  • Sarsi

Code (None):

<ul type="none"> <li>Coke</li> <li>Pepsi</li> <li>Sarsi</li> </ul>

Output:

  • Coke
  • Pepsi
  • Sarsi

6. Description Lists

A description list is a list of terms, with a description of each term.

Code:

<dl> <dt>HTML</dt> <dd>This is the backbone of the website</dd> <dt>CSS</dt> <dd>This is the appearance of the website</dd> <dt>Javascript</dt> <dd>This is the behavior of the website</dd> </dl>

Output:

HTML
This is the backbone of the website
CSS
This is the appearance of the website
Javascript
This is the behavior of the website

7. Special Characters

Using entities to display reserved characters.

Code:

<h2>Is 71 &gt; 13 &amp; 22 &lt; 5?</h2>

Output:

Is 71 > 13 & 22 < 5?

8. Div Blocks with Inline Styles

Using <div> to create block elements with specific dimensions and colors.

Code:

<div style="height: 50px; width: 50px; background-color: red;">Box1</div> <div style="height: 100px; width: 100px; background-color: blueviolet;">Box2</div> <div style="height: 200px; width: 200px; background-color: green;">Box3</div>

Output:

Box1
Box2
Box3
