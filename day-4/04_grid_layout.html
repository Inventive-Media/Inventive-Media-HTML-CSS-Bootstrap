4. Responsive Grid

Use breakpoint prefixes for mobile-first responsive layouts:

<!-- 1 col mobile, 2 cols tablet, 3 cols desktop -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
