CSS Functions

1. calc() - Mathematical Calculations

Width: calc(100% - 40px)
Width: calc(50% - 20px)
Width: calc(100% / 3 - 20px)

calc() allows you to perform calculations for CSS values

2. var() - CSS Custom Properties (Variables)

Uses --primary-color
Uses --secondary-color
Uses --accent-color

var() uses CSS custom properties defined with --property-name

3. min() - Minimum Value

Width: min(500px, 100%)

min() returns the smallest value from a list (resize window to see effect)

4. max() - Maximum Value

Width: max(300px, 50%)

max() returns the largest value from a list

5. clamp() - Responsive Values

Width: clamp(200px, 50%, 600px)

Font Size: clamp(16px, 3vw, 32px)

clamp(min, preferred, max) - keeps value between min and max

6. rgb() / rgba() - Colors

rgb(255, 99, 71)
rgba(52, 152, 219, 0.7)

rgb() and rgba() define colors using red, green, blue values

7. hsl() / hsla() - Colors

hsl(120, 100%, 50%)
hsla(280, 100%, 50%, 0.7)

hsl() uses hue, saturation, lightness for colors

8. linear-gradient() - Color Gradients

Linear Gradient

linear-gradient() creates smooth color transitions

9. radial-gradient() - Radial Gradients

Radial Gradient

radial-gradient() creates circular color transitions

10. url() - External Resources

Background with url()

url() loads external resources like images

11. attr() - HTML Attributes

Hover over me

attr() gets HTML attribute values (shown in tooltip)

12. Combining Functions

Multiple Functions Combined:
calc() + var() + min()

You can combine multiple CSS functions for powerful layouts
