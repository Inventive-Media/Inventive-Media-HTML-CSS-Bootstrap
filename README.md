# HTML and CSS Training Philippines (2026)

A 3-day training course to learn web development basics: HTML, CSS, and Bootstrap 5.

---

## What's Inside

This training is split into **4 courses**. Each course teaches different skills:

- **HTML Course**: HTML Basics
- **CSS Course**: CSS Styling
- **Bootstrap Course**: Bootstrap Framework
- **Tailwind Course**: Tailwind CSS Framework

---

## How to Start

1. Open the `index.html` file in the main folder
2. Click on any course card (HTML, CSS, Bootstrap, or Tailwind)
3. Choose a lesson to learn
4. Use the "Back to Main Home" link to go back

## How to Open HTML Files

### Easy Way (Double-Click)

- Find any `index.html` file
- Double-click it
- It will open in your browser

### Choose Your Browser

- Right-click on `index.html`
- Click "Open with"
- Pick your browser (Chrome, Firefox, Edge, etc.)

### Using VS Code

- Install "Live Server" extension
- Right-click on `index.html`
- Click "Open with Live Server"
- The page will auto-refresh when you make changes
